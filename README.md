# Hextris

A browser-based falling block game inspired by classic block-dropping gameplay. It uses original code and a clean interface while keeping a familiar grid-style layout.

## Play
- Open `index.html` in your browser
- Use arrow keys to move and rotate
- Press `Space` for hard drop
- Press `C` to hold the current piece

## Notes
- This game is not a copyrighted copy of any commercial title.
- The UI is intentionally simple and modern, with a familiar scoreboard and preview panels.
- Firebase auth is included in `script.js`; fill in your Firebase project values in the `firebaseConfig` object.
- Create a Firebase web app and enable Email/Password authentication to use sign-in and sign-up.
